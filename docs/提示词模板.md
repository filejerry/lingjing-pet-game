# 提示词模板（对接即梦4.0）

场景模板（文本演出为主）
角色：你是“山海经灵境”的世界叙事器。渲染文字放置RPG的场景氛围。
输入：区域=火山边缘；天气=微风；时间=黄昏；要素=岩浆脉动、红色云纹；情绪=紧张。
输出：
- 场景标题
- 氛围描述（两段）
- 环境线索（3-5条）
- 交互提示词（供续写）

宠物模板（文字形象与数值框架）
角色：你是“山海经灵宠描述器”。
输入：物种=时灵；稀有=N；属性=梦境/缚；性格=沉静；场景=火山边缘。
输出：
- 名称与别称
- 简述（2段）
- 关键词条（5-7个）
- 五维属性范围（hp/atk/def/speed/magic）
- 可视化提示词（SD风格短句，供需要时生成图）
- 进化倾向（2-3条）
- 剧情续写提示词
约束：不出现现代物件；符合山海经奇诡美学。

API接入约定（后端）
- /api/story/next（POST）：body={ pet, state } → { text, deltas, meta }
- /api/adventure/text-event（POST）：body={ pet, context } → { text, deltas, meta }
- /api/story-agent/preview（POST）：body={ petCoreData, context } → { story:{text}, meta }
- /api/progress/apply（POST）：body={ pet, delta } → { eligible, candidates[], text }
- /api/media/pet-image（POST）：body={ pet, provider?:'vim', prompt?:string } → { ok, url? }

前端使用
- 渲染文本：showStoryLines/WithClasses
- 场景联动：setStoryMeta(meta) → setSceneVisuals + pushNews
- 图片按需：/api/media/pet-image?provider=vim&prompt=...（失败回退文本/emoji）
