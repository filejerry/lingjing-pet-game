# 世界观蓝图（山海经灵境）

核心设定
- 世界树（建木）为世界结构的根，分出多域：青木灵境、东海扶桑、昆仑圣域、幽冥界、龙界、临界之空、赤焰火山带、荒原沙海……
- 灵宠谱系：时灵/星兽/木精/火龙/影魇等，每类具“属性”（元素/梦境/神性/堕落等）与“稀有度”层级。
- 事件类型：daily（日常）、special（奇遇）、side（支线）、battle（对决）、evolution（进化）

区域模块（示例）
- 青木灵境·林径：清风/露水/叶响，情绪=宁静；常见事件=陪伴/练习
- 赤焰火山带：岩浆脉动/红云纹，情绪=紧张；常见事件=试炼/觉醒
- 临界之空：风起云涌/云廊，情绪=肃穆；常见事件=对决/启示
- 荒原沙海：热浪/砂纹，情绪=坚毅；常见事件=探寻/补给

叙事要素模板
- 场景标题、氛围两段、环境线索（3-5）、交互提示词（供续写）
- 宠物文字形象：名称/别称、简述两段、关键词条（5-7）、五维范围、进化倾向（2-3）、续写提示词

术语词典（摘）
- 羁绊：bond；灵识：spirit sense；临界：threshold；觉醒：awakening；堕落：fall
