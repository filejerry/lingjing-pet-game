# 🐾 灵境斗宠录

AI驱动的文字宠物养成游戏 - 每个选择都将影响你宠物的命运

## ✨ 游戏特色

- 🤖 **AI智能驱动**: 三层AI架构，让每只宠物都有独特的进化路径
- 🐉 **山海经主题**: 丰富的中国神话生物，从普通到传说级稀有度
- 🎮 **选择式交互**: 黑白简约UI，通过选择推进剧情
- 💬 **智能对话**: 与宠物自由聊天，体验AI互动
- ⚔️ **战斗系统**: 多人在线对战，展示宠物实力
- 🎨 **稀有度系统**: 七彩渐变的SSS级传说宠物

## 🎯 游戏玩法

1. **创建宠物**: 选择你理想的伙伴类型
2. **养成互动**: 通过喂食、探索、训练来培养宠物
3. **AI进化**: 宠物会根据你的行为智能进化
4. **战斗对决**: 与其他玩家的宠物进行战斗
5. **收集养成**: 收集不同稀有度的神话生物

## 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (黑白简约风格)
- **后端**: Node.js + Express
- **数据库**: SQLite (本地) / PostgreSQL (生产)
- **AI引擎**: 三层智能决策系统
- **部署**: Vercel + GitHub

## 📱 功能特性

### 🐉 宠物系统
- **稀有度等级**: N(普通) → R(稀有) → SR(超稀有) → SSR(极稀有) → SSS(传说)
- **属性系统**: 生命、攻击、防御、敏捷、魔法
- **进化机制**: AI驱动的智能进化系统
- **个性化**: 每只宠物都有独特的性格和能力

### 🎨 视觉设计
- **黑白主题**: 简约优雅的界面设计
- **稀有度颜色**: 
  - 普通(灰色) → 稀有(蓝色) → 超稀有(紫色) → 极稀有(红色) → 传说(七彩渐变)
- **动画效果**: 流畅的过渡和悬停效果
- **响应式**: 完美适配各种设备

### 🤖 AI系统
- **L1层**: 行为记录层，记录所有玩家操作
- **L2层**: 进化判断层，决定宠物进化方向  
- **L3层**: 属性固化层，将AI决策转为游戏数值
